Gulp starter kit by Daniel Yates.
Utilises Jade, SCSS, JS, Gulp task runner.

## Gulp will:
1. Compile and compress your Jade files into HTML
2. Compile and compress your SASS/ SCSS into CSS
3. Compile and minify all your JS
4. Compress all images
5. Allow you to preview changes asynchronously in the browser
6. Allow you to connect to any device on the network to develop asynchronously

## To use:
1. Open Terminal
2. cd into project directory
3. Run ‘gulp’
4. Make changes!!!
